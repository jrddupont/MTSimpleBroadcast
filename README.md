# MT Simple Broadcast
This is a slimmed down version of [EspMeshCompact](https://github.com/htotoo/EspMeshCompact) designed to provide simple broadcast functionality for Meshtastic.   
It is very experimental and should be used with caution.

## Example Usage
### Environment
Requires these libraries:
* jgromes/RadioLib
* rweather/Crypto
* densaugeo/base64  
  
Examples:  
* [Raspberry Pi Pico](https://github.com/jrddupont/PiPico-MeshPinger)
* [Heltec V3](https://github.com/jrddupont/HeltecV3-MeshPinger)
